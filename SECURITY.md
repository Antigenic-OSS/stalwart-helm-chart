# Security Policy

## Reporting a vulnerability

Please do not open public issues for suspected vulnerabilities.

Report security concerns privately to:
- Antigenic <oss@antigenic.org>

Include:
- Affected chart version
- Reproduction details
- Impact assessment
- Any known mitigation

## Response targets

- Initial acknowledgement: within 3 business days
- Triage status update: within 7 business days

## Scope notes

This project is an unofficial Helm chart and does not control upstream Stalwart Mail Server code. Reports that affect upstream Stalwart will be coordinated as appropriate with upstream maintainers.
